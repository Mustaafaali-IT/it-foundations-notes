Today I learned how to initialize and connect Git BASH to my public GitHub profile to allow me to make and save changes locally. I can then commit and push changes to Github



**Commands**

* cd - change directory
* ~ - main directory
* mkdir - make directory
* git - use git
* clone - make a copy of a remote repository
* ls - list
* git status - see what's changed
* git add - staging
* git commit -m - explain changes with message (-m)
* git push - publish
* git add . - everything in the folder (.)
